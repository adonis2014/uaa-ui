# uaa-ui
Web UI for Cloud Foundry UAA


```
./mvnw clean package -DskipTests=true

java -jar target/uaa-ui-0.0.1-SNAPSHOT.jar
```

Go to [http://localhost:8080](http://localhost:8080)
